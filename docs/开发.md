开发工具: PyCharm

python版本: python:3.8

启动前先从example/configs目录拷贝application.yml文件到application.yml文件

根据依赖文件安装依赖

```
pip install -r requirements.txt
```

导出本地依赖到文件

```
pip freeze > requirements.txt
```

开发过程中安装的依赖

```
# 安装flask网络框架
pip install Flask
pip install urllib3
pip install Flask-Cors

# 安装yaml
pip install PyYAML

# 安装dbpool
pip install DBUtils

# 安装mysql
pip install PyMySQL
pip install requests
```



logs和temp目录为临时目录, 需要挂载到持久目录中
